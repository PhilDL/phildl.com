Manywalls is a digital exhibition platform for artists and visual work.

I built it as a richer alternative to static portfolio sites, with mobile-first
experiences, image-heavy interactions, and a backend that supports real-time
product behavior. The project also pushed work on design systems and reusable
editing primitives.
